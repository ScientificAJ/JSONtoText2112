# JSONtoText2112
Hi, This tool converts .json files to .txt file.

# Usage:

```
$ chmod +x JSONtoText2112.py

$ ./JSONtoText2112.py


```

or

```

$ python3 JSONtoText2112.py

```

# Installation:

```

$ git clone https://github.com/ScientificAJ/JSONtoText2112.git

$ cd JSONtoText2112/

$ chmod +x JSONtoText2112.py

```
